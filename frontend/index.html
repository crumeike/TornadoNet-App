Deploy locally
  1. 1Clone the repository: git clone https://github.com/crumeike/TornadoNet
  2. 2Install dependencies: pip install ultralytics fastapi uvicorn python-multipart
  3. 3Place checkpoint files in checkpoints/ or set a HuggingFace token for auto-download
  4. 4Start the server: uvicorn api.main:app --reload --port 8000
  5. 5Enter http://localhost:8000 above and click Test connection
